wasm_builder live demo

wasm_builder is a utility for programmatically building WASM modules in JavaScript. It is a single file that runs in the browser and NodeJS.

Open the web console to view the output of the demo file. If you see "Hello World!", then the code ran successfully.

To start a new module, type the following into the console:

const new_mod = WASMBuilder.module();

For a usage guide, please see the GitHub repo.
